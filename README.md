# Bike Sales Analysis Dashboard (Excel)

## Project Overview
This project analyzes customer purchasing behavior for bike sales using Microsoft Excel. The dashboard provides interactive insights into demographic and behavioral factors influencing purchase decisions.

## Business Questions Addressed
- Which customer segments are most likely to purchase bikes?
- How does income influence purchase behavior?
- What age group shows the highest purchase rate?
- How does commute distance affect buying decisions?
- Does car ownership impact bike purchases?

## Dashboard Features
- Interactive slicers (Marital Status, Region, Education)
- Occupation-based purchase segmentation
- Gender income comparison
- Age bracket analysis
- Commute distance trends
- Car ownership impact

## Dataset Summary
- Total Records: 1000 customers
- Purchase Variable: Yes / No
- Key Attributes: Gender, Age, Income, Occupation, Region, Education, Cars Owned, Commute Distance

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Interactive Slicers
- Data Aggregation & Segmentation

## Key Skills Demonstrated
- Business Data Analysis
- Dashboard Design
- Customer Segmentation
- Data Visualization
- Analytical Thinking

## How to Use
1. Open the Excel file.
2. Navigate to the **Dashboard** sheet.
3. Use slicers to filter by region, education, or marital status.
4. Analyze customer segments and purchasing patterns.
